h2. Redmine issue id's query plugin

Plugin adds extra operators to issues query, allowing to select exact issues by their id's (comma-separated list).

h3. Compatibility

Plugin was tested on Redmine v1.4.1.stable (MySQL)

h3. History

0.0.1

* First public release

Plugin was tested on Redmine v1.3.2.devel (r9308)

0.0.2

* Add new filter option, for excluding issues by id
* Backward compatibility with old filters
* Delete view template patch (_filters.html.erb)

0.0.3

* Redmine 2.x support
